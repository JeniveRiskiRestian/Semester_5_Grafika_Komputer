# a. Judul Proyek
Tupai Courier
Game 2D sederhana berbasis Python sebagai proyek Ujian Tengah Semester (UTS) Grafika Komputer Semester 5.

# b. Konsep Grafika yang Digunakan
Program ini menerapkan beberapa konsep grafika komputer 2D, yaitu: Grafika 2D berbasis koordinat kartesius. Transformasi Geometris 2D, meliputi:
Translasi (perpindahan objek)
Rotasi (perputaran objek musuh)
Skala (perubahan ukuran player)
Refleksi (pembalikan arah player)
Rendering manual objek grafis tanpa menggunakan fungsi gambar bawaan Pygame

# c. Algoritma yang Dipakai
Algoritma grafika yang diimplementasikan secara manual dalam program ini antara lain:
Algoritma Garis DDA (Digital Differential Analyzer)
Digunakan untuk menggambar ground, health bar, dan sisi poligon.
Algoritma Lingkaran Midpoint
Digunakan untuk menggambar boomerang dan efek partikel.
Algoritma Poligon
Digunakan untuk membentuk objek musuh dan power-up.
Algoritma Transformasi Geometri 2D
Untuk translasi, rotasi, skala, dan refleksi objek.

# d. Cara Menjalankan Program
Pastikan Python 3.x sudah terinstall.
Install library Pygame dengan perintah: pip install pygame
Clone repository ini: git clone [URL_REPOSITORY]
Masuk ke folder project dan jalankan program: python TupaiCourier.py
